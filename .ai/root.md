# Root AI Portal

This folder contains the master rules and architecture documents for the Artha system.
Modern agents should start here.

- `architecture.md`: System-wide architecture — data flow, pipelines, auth, infra, ADRs.
- `coding-standards.md`: Global coding standards (async patterns, error handling, config).
- `backend.md`: Backend (Python/FastAPI) structure and DI rules.
- `frontend.md`: Frontend (Next.js) structure and Server Component rules.
- `database.md`: Database schemas, pgvector/IVFFlat configuration, all ORM models.
- `agents.md`: LangGraph agent patterns — RAG pipeline, ingestion pipeline, quality gates.
- `workflows.md`: CLI cheat sheets, development workflows, observability.

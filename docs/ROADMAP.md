# Agentic RAG Roadmap 🗺️

This document outlines the planned features and improvements for Agentic RAG.

## 🚀 Phase 1: Core Foundation (Current)
- [x] Local LLM integration (Ollama)
- [x] Vector storage with pgvector
- [x] Async document ingestion (PDF, DOCX, TXT)
- [x] Agentic retrieval with LangGraph
- [x] Real-time streaming UI
- [x] JWT Authentication

## 🛠️ Phase 2: Enhanced Capabilities (Q2 2026)
- [ ] **Multi-Model Support**: Native support for Anthropic, OpenAI, and Groq via LangGraph.
- [ ] **Advanced Ingestion**: Support for Excel, Markdown, and Web URL scraping.
- [ ] **Citation Highlighting**: Direct links from citations to document snippets in the UI.
- [ ] **Conversation History Search**: Semantic search over past conversations.
- [ ] **Export Options**: Export chat history and project data to PDF/JSON.

## 🛡️ Phase 3: Enterprise & Scale (Q3 2026)
- [ ] **Role-Based Access Control (RBAC)**: Admin vs. Member permissions.
- [ ] **Self-Correction (Agentic RAG+)**: Implementing "Corrective RAG" (CRAG) patterns.
- [ ] **Multimodal RAG**: Support for image ingestion and analysis.
- [ ] **Deployment Templates**: Kubernetes Helm charts and Terraform scripts.
- [ ] **Telemetry**: Deeper integration with Prometheus and Grafana.
